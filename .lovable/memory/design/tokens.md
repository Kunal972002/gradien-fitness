## Color Palette (oklch)
- Emerald (primary): oklch(0.42 0.14 160) — headers, trust elements, active states
- Emerald light: oklch(0.92 0.06 160) — subtle backgrounds
- Emerald dark: oklch(0.28 0.1 160) — footer background
- Citrus (CTA): oklch(0.7 0.18 55) — Book Call buttons, WhatsApp, high-priority actions
- Citrus hover: oklch(0.65 0.19 55)
- Warm gray: oklch(0.97 0.003 260) — section backgrounds
- Background: pure white oklch(1 0 0)
- Foreground: deep charcoal oklch(0.15 0.01 260)

## Typography
- Headings: Montserrat (--font-heading), bold/black, uppercase for main titles
- Body: Inter (--font-sans)

## Button Variants
- citrus: CTA buttons with shadow and hover lift
- emerald: trust/secondary actions
- outline: border-2, inverts on hover
